# Visual Guide - STRIDE Website Sections

This guide describes what each section of the website looks like and how it functions.

## 🎯 Navigation Bar (Top)

**Location:** Fixed at top of page

**Visual Elements:**
- Left: "STRIDE" logo in gradient colors
- Center: Navigation links (Home, Collection, Features, Gallery, Contact)
- Right: Dark mode toggle (sun/moon icon), Shopping bag icon, Mobile menu button

**Behavior:**
- Transparent initially, becomes solid white/dark on scroll
- Smooth scroll to sections when clicking links
- Mobile: Hamburger menu that slides down
- Hover effects on all clickable elements

## 📱 Responsive Behavior

**Mobile (< 768px):**
- Single column layouts
- Hamburger menu
- Stacked cards
- Larger touch targets
- Simplified animations

**Tablet (768px - 1024px):**
- 2-column layouts
- Visible navigation
- Medium-sized images
- Balanced spacing

**Desktop (> 1024px):**
- Multi-column layouts
- Full navigation
- Large images
- Enhanced animations
- Hover effects

---

## ✨ Animation Highlights

1. **Page Load:** Elements fade in and slide up
2. **Scroll:** Sections animate into view
3. **Hover:** Cards lift, buttons scale, colors change
4. **Click:** Feedback animations, smooth transitions
5. **Background:** Gradients rotate, patterns move
6. **Products:** Float, rotate, zoom effects
7. **Countdown:** Numbers flip, timer updates
8. **Slider:** Smooth slide transitions
